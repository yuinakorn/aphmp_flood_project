# ระดับความเร่งด่วน: P1/P2/P3 vs A/B/C

สองระบบนี้วัดคนละมิติ — อย่าสับสนกัน

---

## P1 / P2 / P3 — Evacuation Priority (EOC Command Queue)

**ที่อยู่ใน UI:** แท็บ "คิวสั่งการ" หน้า EOC (`/admin/eoc`)

**ความหมาย:** ลำดับความเร่งด่วนที่ระบบ**คำนวณอัตโนมัติ**ว่าควรส่งทีมไปรับ**ครัวเรือน**ไหนก่อน — เป็น logistics/evacuation priority ไม่ใช่ medical triage

**สูตรคำนวณ** (`src/lib/overview.ts` → `getOverviewData`):

```
score = 0.4 × fLife + 0.3 × fProx + 0.12 × fContact + 0.10 × fNoCare + 0.08 × fReq
```

| ตัวแปร | ความหมาย | น้ำหนัก |
|--------|----------|---------|
| `fLife` | อุปกรณ์พยุงชีพ (O2=1.0, ฟอกไต=0.9/0.85, ยากันชัก=0.6, สายให้อาหาร=0.5) | 40% |
| `fProx` | ความใกล้น้ำ (flood=1, near=0.5, safe=0) | 30% |
| `fContact` | ไม่ได้ติดต่อนานแค่ไหน (cap ที่ 48 ชม.) | 12% |
| `fNoCare` | ไม่มีผู้ดูแล (1=ไม่มี, 0=มี) | 10% |
| `fReq` | มีคำร้องช่วยเหลือค้างอยู่ | 8% |

**เกณฑ์ตัดสิน:**

| ระดับ | score | ความหมาย |
|-------|-------|----------|
| **P1** | ≥ 0.6 | วิกฤต — ต้องส่งทีมก่อน |
| **P2** | 0.35–0.59 | เร่งด่วน |
| **P3** | < 0.35 | เฝ้าระวัง |
| **unknown** | — | ข้อมูลไม่ครบ (confidence < 34% และไม่มี life support และ safe) |

**หน่วยวิเคราะห์:** ครัวเรือน (household) — ถ้ามีสมาชิกหลายคน ใช้ worst-case ของครัวเรือน

---

## A / B / C — Medical Priority (Map Roster)

**ที่อยู่ใน UI:** แผง Roster บนหน้าแผนที่ (`/map`) — badge สีแดง/ส้ม ข้างชื่อ

**ความหมาย:** ระดับความวิกฤต**ทางการแพทย์** กรอกโดยเจ้าหน้าที่ (อสม./พยาบาล) ตอนเยี่ยมบ้าน — เป็น clinical judgment ไม่ได้คำนวณ

| ระดับ | ความหมาย |
|-------|----------|
| **A** | Critical — ต้องรักษาทันที |
| **B** | Urgent — เร่งด่วน |
| **C** | ไม่วิกฤต / ติดตาม |

**field ในฐานข้อมูล:** `household_members.medical_priority` (type: `MedicalPriority = 'A' | 'B' | 'C'`)

---

## ตัวอย่างความสัมพันธ์

คนคนเดียวกันสามารถมีค่าสองระบบที่ดูขัดกันได้:

| กรณี | P | Medical |
|------|---|---------|
| อยู่ในน้ำท่วม + พึ่ง O2 + ไม่มีผู้ดูแล | P1 | A |
| บ้านใกล้น้ำแต่ยังไม่ท่วม + แข็งแรงดี | P2 | C |
| อยู่ที่ปลอดภัยแต่ป่วยหนัก | P3 | A |

> กรณีสุดท้าย: P3 แต่ Medical A → ระบบไม่ได้บอกให้ไปรับก่อน แต่เจ้าหน้าที่ควรดูช่อง Medical ประกอบด้วย
